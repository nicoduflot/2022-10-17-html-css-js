Vidéos hébergées

Une vidéo, un format

Au niveau de l'accessibilité, une vidéo doit :

  • - Toujours posséder des sous-titres
  • - Si possible une transcription de ce qui est dit disponible en un clic
  • - éviter l'autoplay sur les vidéo

une vidéo, plusieurs formats

Format : taille d'affichage, extension (.webm, .mp4, mkv, etc...)

Une même vidéo peut-être proposée sous différents fichiers, formats et c'est le navigateur qui déterminera le fichier le plus adapté (carte vidéo, connexion internet, etc)
